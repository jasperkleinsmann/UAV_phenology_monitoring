# Monitoring Individual Tree Phenology in a Multi-Species Forest using High Resolution UAV Images

This repository contains all the scripts developed for the study: **"Monitoring Individual Tree Phenology in a Multi-Species Forest using High Resolution UAV Images"**.

The following link contains some of the data used for the study (DOI). The minimal required data to run the scripts has been stored in `datasets/data`. To allow users to run parts of the analysis without extensive computation times, some intermediate datasets have been made available as well in the `datasets/output` folder.
